jQueryの基本書式

h1を隠す
        $(function() {
          $('h1').hide();
        });
h1を徐々に（1500ミリ秒かけて）隠す
        $(function() {
          $('h1').fadeOut(1500);
        });
メソッドの後の（）内には引数が入り、速度を指定するなどできる
指定したIDの要素を徐々に表示させる
        $(function() {
          $('#id-name').fadeIn('slow');
        });
引数には数値だけでなくslowのようなキーワードも指定できる。キーワードは''で囲う。
ボタンをクリック（イベント発生）すると、コードが現れる
クリックしてコードを表示
          $('#appear').click(function(){
            $('#click-fadeIn').fadeIn();
        });
